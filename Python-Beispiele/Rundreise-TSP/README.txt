Die beste gefunden Distanz ist 22239 km

Die Simulation kann mit der folgenden Kommandozeile gestartet werden.

python Rundreise.py cities.csv plot 100 -1 0.93 1600 .5 -1

oder Ausgabe in Datei

python Rundreise.py cities.csv save 100 -1 0.93 1600 .5 -1

Die Simulation kann jeder Zeit mit ctrl-x gestoppt werden.

Parameter:
python Rundreise.py input [plot] [iterations] [nb_cities] [c_factor] [t_start] [t_end] [r_seed]
